# 02 — Timeline

This project took us approximately 3 weeks from the start to the delivery.

## Timeline

| Stapes | Target | Deliverable |
| --------- | ------ | ----------- |
| 1 - Setup | Week 1, day 1 | Repo, tooling, task split, Trello board |
| 2 - Config & maze | End of Week 1 | Validated config + maze rendered on screen |
| 3 - Playable core | Mid Week 2 | Player moves, eats Pacgums, ghosts move |
| 4 - Full game loop | End of Week 2 | Lives, timer, levels, win/lose, highscores |
| 5 - Polish & bonus | Week 3 | Themes, cheat mode, UI scaling, lint clean |
| 6 - Delivery | End of Week 3 | Docs, PM folder, final review |



## Workflow (Trello)

The Trello board was the single source of truth for day-to-day work. In this tool, we made a card for all the tasks we had to implemente.
We tagged ourselves on the card when we worked on it, and validated it when the task was finished.
